🛒 E-Commerce Customer Analysis

📌 Project Overview

This project analyzes e-commerce customer data to understand business performance, customer behavior, and revenue trends. The project covers the complete data analysis pipeline from data cleaning to machine learning.

---

🎯 Objectives

- Analyze revenue and profit performance
- Identify top-performing product categories
- Understand customer behavior and segmentation
- Analyze payment methods and order status
- Predict future revenue trends

---

🛠️ Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (PostgreSQL)
- Power BI
- Machine Learning (KMeans, Linear Regression)

---

🔄 Project Workflow

1. Data Cleaning
   
   - Removed duplicates
   - Handled missing values
   - Fixed date formats
   - Cleaned and standardized data

2. Data Analysis (SQL & Python)
   
   - Calculated KPIs (Revenue, Profit, Orders, Rating)
   - Category-wise performance analysis
   - Payment method analysis
   - Monthly revenue trend

3. Data Visualization
   
   - Built dashboard using Power BI & Python
   - KPI cards, bar charts, pie charts, trend analysis

4. Machine Learning
   
   - RFM Analysis for customer segmentation
   - KMeans clustering for customer types
   - Linear Regression for revenue prediction
   - Forecasting future revenue

---

📊 Key Insights

- Total Revenue: ₹27.53M and Total Profit: ₹12.39M
- Customer rating is low (3.01), indicating improvement needed
- Few categories generate majority of revenue
- Digital payment methods are widely used
- Top customers contribute a large portion of revenue (Pareto Principle)
- Sales trend shows fluctuations, indicating inconsistent growth
- Customer segmentation helps in targeted marketing
- Revenue prediction and forecasting support business planning
